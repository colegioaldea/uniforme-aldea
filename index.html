Crear nuevo año
Cierre de año y apertura de nuevo período
Al crear un nuevo año:
• Las ventas y compras actuales se archivan en el historial
• El stock actual de cada producto pasa a ser el stock inicial del nuevo año
• El PV de cada producto se actualiza con el último precio de venta registrado (PV3 → PV2 → PV1 → PV)
• Los campos PC1…PV3 se limpian para el nuevo período
• Las entradas y salidas se resetean a 0
